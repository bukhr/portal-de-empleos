# Portal de Empleos

Plataforma pública que centraliza las ofertas laborales publicadas por empresas que utilizan Buk, permitiendo a postulantes externos descubrir y postular fácilmente a oportunidades laborales.

* Ruby version: 3.3.8
* System dependencies
* Configuration
